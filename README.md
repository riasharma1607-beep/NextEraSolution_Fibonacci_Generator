# Fibonacci Generator

This project is created as part of the NextEra Solution With Technology Internship Program.

## Description
This Python program generates the Fibonacci series based on user input.
The Fibonacci series is a sequence where each number is the sum of the two preceding numbers.

## Technologies Used
- Python
- Google Colab

## How to Run
1. Open the file in Google Colab or any Python environment
2. Run the cell and enter the number of terms
